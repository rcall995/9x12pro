## Local Development

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment variables:**
   - Copy `.env.example` to `.env.local`
   - Add your Supabase credentials

3. **Run locally:**
   ```bash
   npm run dev
   ```
   Open http://localhost:3000

## Deploy to Vercel
